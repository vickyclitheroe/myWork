Wordpress Project

This is a clients site I worked on while completing an internship in Brighton.

The client has a medical recruitment company which already had a site set up, she wanted a bit of a "facelift" done to the website.

The Wordpress Universe theme was chosen as a base for the site

I had to take the site from looking like this theme to this.

From Theme to customised site

Responsive WebSite

Some tasks I completed during this were :

  • Css-Restructuring the layout and design such as the colours and sizes
  • Content-adding all new pages, images and content.
  • Responsiveness- Making site responsive on other browsers and mobile.
  • Wordpress admin-plugins,menus,widgets,contact forms etc.
  • Some photoshop editing on images and logo.

I really enjoyed working on this Wordpress site. It was very satisfying to see the site develop and end up looking exactly like what the client had in mind.
